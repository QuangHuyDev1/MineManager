## 🚀 MineManager Update 1.1

MineManager là một trình khởi chạy (Launcher) Minecraft hiện đại, mượt mà và tích hợp sẵn các công cụ tối ưu hóa hệ thống mạnh mẽ. Được thiết kế với giao diện Neon đặc trưng, MineManager không chỉ giúp bạn quản lý game mà còn nâng tầm trải nghiệm chơi game trên các dòng máy cấu hình tầm trung.

✨ Tính năng nổi bật
🎮 Quản lý Game chuyên nghiệp
Khởi động nhanh: Hỗ trợ đầy đủ các phiên bản từ Snapshot đến chính thức.

Quản lý cấu hình (Profiles): Tạo nhiều profile riêng biệt (Vanilla, Modded, Performance) với cài đặt RAM và phiên bản Java khác nhau.

Cài đặt Mod tự động: Tích hợp bộ lọc Mod từ Modrinth (hỗ trợ cả Fabric và Forge).

Trình quản lý Server: Danh sách máy chủ phổ biến (Hypixel, Mineplex, CubeCraft) sẵn sàng kết nối.

⚡ Tối ưu hóa hệ thống (Built-in Optimizer)
Điểm khác biệt nhất của MineManager so với các Launcher thông thường:

Tích hợp AtlasOS: Hỗ trợ cài đặt và tối ưu hóa Windows thông qua các bản script "debloat" chuyên sâu.

Ultimate Power Plan: Kích hoạt chế độ năng lượng tối đa để duy trì FPS ổn định.

Game Mode: Tối ưu hóa các tiến trình nền trước khi vào game.

Theo dõi tài nguyên: Hiển thị trực quan tình trạng CPU, RAM, GPU và dung lượng ổ đĩa ngay trên Dashboard.

🎨 Giao diện & Trải nghiệm (UI/UX)
Thiết kế Neon: Giao diện Dark mode hiện đại với các hiệu ứng phát sáng (Glow effect).

Cá nhân hóa: Cho phép thay đổi Theme và điều chỉnh phân bổ RAM (Memory Allocation) linh hoạt.

Screenshot Gallery: Quản lý và xem lại những khoảnh khắc trong game một cách dễ dàng.
